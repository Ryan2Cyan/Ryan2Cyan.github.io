Vector Length

Material Function Node

Summary:

Returns the length/magnitude of a 2D or 3D vector (how far it is from the origin [0, 0]). This node can be used to create a radial shape.

Node Breakdown:

Vector 3 (Vector3): Vector3 input.

Vector 2 (Vector2): Vector2 input.

V3 Length 3 (Vector3): The length of Vector 3 input.

V2 Length (Vector2): The length of Vector 2 input.

Mathematics

The mathematical formula to calculate a vectors length is:

Vector2
Vector3

sqrt(x * x + y * y)
sqrt(x * x + y * y + z * z)

Example: We have a 2D coordinate where x = 0.77 and y = -0.12. We need to find the length of this 2D vector [0.77, -0.12] from the origin [0, 0]:

xSqr = x * x
ySqr = y * y
SumXY = xSqr + ySqr
Length = sqrt(SumXY)
Result

0.5929 = 0.77 * 0.77
0.0144 = -0.12 * -0.12
0.6073 = 0.5929 + 0.0144
0.78 = sqrt(0.6073)
